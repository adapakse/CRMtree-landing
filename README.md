# CRMtree Landing Page

Static landing page for [crmtree.pl](https://crmtree.pl), hosted on Cloudflare Pages.

## Deploy

Cloudflare Pages → Connect to Git → this repo  
Build settings: framework = None, output directory = `/`

## Custom domain

Add `crmtree.pl` and `www.crmtree.pl` in Cloudflare Pages → Custom domains.

## Contact form

Uses Formspree. Replace `REPLACE_WITH_YOUR_ID` in `index.html` with your Formspree form ID.
Create free account at https://formspree.io — up to 50 submissions/month free.
